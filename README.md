# 📖 Receptbok

En webbapplikation där användaren kan lägga till, redigera och ta bort recept.

## Funktioner

-  Lägg till recept med titel, kategori och beskrivning
-  Redigera befintliga recept
- Ta bort recept
- Sök recept på titel eller kategori
- Datum visas när receptet skapades och senast redigerades
- Recept sparas i localStorage
- Responsiv design – fungerar på mobil och desktop

## Tekniker

- HTML
- CSS
- JavaScript (vanilla)
- localStorage

## Kom igång

1. Klona repot
2. Öppna `index.html` i webbläsaren
