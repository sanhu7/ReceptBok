# 📖 Receptbok

En webbapplikation byggd med vanilla JavaScript där användaren kan 
hantera sina recept( lägga till, redigera och ta bort recept).

## Funktioner

-  Lägg till recept med titel, kategori och beskrivning
-  Redigera befintliga recept-klicka på "Redigera" för att ändra ett befintligt recept.
- Ta bort recept- klicka på "Ta bort" för att radera ett recept.
- Sök recept - filtrera listan i realtid på titel eller kategori.
- Datum- visas när receptet skapades och senast redigerades
- localStorage- recept sparas automatiskt och finns kvar vid omladdning
- Responsiv design – fungerar på alla skärmstorlekar från 360px och uppåt.

## Kategorier

Recepten kan kategoriseras som:
- Frukost
- Lunch
- Middag
- Dessert
- Snacks
